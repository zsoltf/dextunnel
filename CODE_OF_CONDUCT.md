# Code Of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Our Pledge

We want Dextunnel to be a respectful, constructive project for everyone who participates.
Contributors, maintainers, and community members should aim to make collaboration welcoming, thoughtful, and technically honest.

## Our Standards

Examples of behavior that help create a positive environment include:

- showing empathy and respect
- giving and receiving feedback constructively
- focusing on what is best for the project
- assuming good faith while still being clear about problems
- respecting different viewpoints, experience levels, and backgrounds

Examples of unacceptable behavior include:

- harassment, intimidation, or discrimination
- personal attacks, trolling, or insulting language
- publishing private information without permission
- sustained disruptive behavior in issues, pull requests, or discussion spaces
- sexualized language or imagery

## Scope

This Code of Conduct applies within project spaces and in public spaces when someone is representing the project or its community.

## Reporting

For Code of Conduct concerns, please contact the maintainers through a private channel rather than opening a public issue.

For security-sensitive reports, use the process in `SECURITY.md`.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct and may remove, edit, or reject comments, commits, code, wiki edits, issues, and other contributions that are not aligned with it.
